## Play-note ##

The project allows users to share youtube playlists. 

This project mainly contains these functions:

-User registration
-User login
-Playlist upload
-Adding playlist to favorites
-Removing playlist from favorites
-Adding likes
-Removing likes

Technologies used:

Laravel 4.2, bootstrap, pure css 3.0.
